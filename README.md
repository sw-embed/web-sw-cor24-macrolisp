# web-tml24c

Web UI for [Tiny Macro Lisp](https://github.com/sw-vibe-coding/tml24c) on COR24. Live browser demos of the Lisp system running on the COR24 emulator.

## Related

- [tml24c](https://github.com/sw-vibe-coding/tml24c) -- The Lisp implementation
- [tc24r](https://github.com/sw-vibe-coding/tc24r) -- COR24 compiler in Rust
- [cor24-rs](https://github.com/sw-embed/cor24-rs) -- COR24 assembler and emulator
